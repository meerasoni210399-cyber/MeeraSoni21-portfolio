## ✨ Features

* **Glassmorphism UI:** Trendy and clean frosted-glass effect with smooth gradients.
* **Interactive Dashboard:** Quick view of skills, projects, and certifications all on one screen.
* **Project Carousel:** Sleek and dynamic sliders to browse through projects and certificates easily.
* **Responsive Design:** Fully optimized and functional across all screen sizes and devices.

---

## 🛠️ Tech Stack

This project was built using the following technologies:

* **HTML5** - For semantic structure and layout markup.
* **CSS3** - For glassmorphism effects, custom animations, Flexbox, and Grid layouts.
* **JavaScript** - For dynamic data rendering, real-time date/time display, and carousel functionality.
